# 🔍 GitHub Finder

Une application web pour rechercher des profils GitHub, consulter leurs repositories et sauvegarder des favoris.

---

## ✨ Fonctionnalités

- 🔎 Rechercher un utilisateur GitHub par username
- 👤 Afficher le profil : avatar, bio, followers, repos
- 📦 Lister les repositories triés par étoiles
- ⭐ Ajouter / Retirer des favoris
- 💾 Sauvegarde des favoris dans le localStorage
- 📱 Interface responsive (mobile + desktop)

---

## 🛠️ Technologies

| Technologie | Usage |
|-------------|-------|
| HTML5 | Structure |
| CSS3 | Style + Responsive |
| JavaScript (Vanilla) | Logique + DOM |
| GitHub REST API | Données utilisateurs |
| localStorage | Persistance des favoris |
